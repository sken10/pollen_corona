六角形の花粉光環の数値シミュレーション#

朝夕の太陽高度が低いとき、スギ花粉による花粉光環が六角形になることがある。 この現象に、スギ花粉の突起(パピラ)が関係することを確認した(図1-1, 1-2)。 比較のためパピラ無し結果も示す(図1-3)。 解析は数値シミュレーションによる。 解析に使ったプログラムを (末尾) に示す。 なお別稿にて、小さなパピラが花粉光環の形に大きく影響する仕組みについて、 位相の可視化を用いて直感的な説明を与えた(パピラの影響力の直感的な理解)。

計算モデル#

開口面の複素振幅が \(u(x, y)\) の時、 像面の複素振幅 \(U(x_q, y_q, z_q)\) は以下の式で与えられる (フレネル・キルヒホッフ/Fresnel Kirchhoff の回折積分, \(\lambda\) は波長)。 シミュレーションでは、\(u(x,y) = 1:開口部, 0:遮蔽部\) として、 この回折積分をそのまま計算した。 計算量は多いが理論と実装の単純さを優先した結果である。 計算量が O(N^4) であり、筆者の環境では N_SEP は 100 くらいが限界。

\[ \begin{align}\begin{aligned}U(x_q,y_q,z_q) = \frac{z_q}{i\lambda} \iint_\Sigma u(x,y)\frac{1}{r^2}exp(i\frac{2\pi r}{\lambda}) dx dy\\r = \sqrt{(x_q - x)^2 + (y_q - y)^2 + z_q^2}\end{aligned}\end{align} \]

六角形の花粉光環に関するあれこれ#

朝夕の太陽高度が低いときに、 花粉光環が縦長の楕円になる(場合がある)ことは 以前から知られていたが、 六角形(多角形)になる(場合がある)ことが指摘されたのは比較的最近のことである。 筆者の調べでは Twitter 上で「花粉光環 + 六角形」の両方を含む投稿は、 2022年3月の投稿が最初である。

皆が不思議に思っていたのは、 ほぼ球形といわれるスギ花粉によって六角形の花粉光環が現れることだった。 花粉が完全な球ではなくつぶれていれば(例えば未熟な状態)、 花粉光環が楕円になることは説明できそうだ。 しかし、それが角張って六角形になる理屈は不明のままであった。

2023年になると、 パピラと呼ばれる小さな突起と六角形の花粉光環の関連を指摘する人物が現れた。 その手法は花粉形状を FFT にかけて六角形の出現を確認したものだった。 同じ時期に、FFT を使わずに回折積分を直接解くことで同様の結果を得ていた筆者は、 仮説を補強する別解として twitter に解析の結果を投稿した。

(以上は、twitter という狭い範囲の見分にもとづく簡単なまとめであり、 専門家にとってはあたりまえとう可能性は大いにあり得る。)

海外事例では、 太陽高度が低いときにマツ花粉によって菱形の花粉光環が出現することが知られており、 その理由がマツ花粉の特異な形状によることも解明されている [1]。 今回意外だったのは、スギ花粉の小さなパピラが花粉光環の形に大きく影響することだった。 別稿でその仕組みについて直感的な説明を試みた。そちらも合わせて参照されたい(パピラの影響力の直感的な理解)。

注意点など#

このレポートでは数値シミュレーションの結果だけを示している。 実際に六角形の花粉光環が現れた時の条件について、 別途検討や確認する必要がある。例えば、

  • 花粉のパピラがしっかりしている

  • 大気中で花粉の向きが揃っている(無風が良さそう。パピラは上で安定か)

  • 花粉を横から見る(太陽高度が低い朝夕に限られる)、

  • 花粉の熟し具合やいびつさ(花粉がつぶれていると花粉光環が縦長になる)

花粉粒子は Mie散乱の適用範囲よりも十分に大きいが、微細なパピラだけ取り出すと、 ちょうど Mie散乱の適用範囲の境界付近となる。そのあたりは気になるところ。

最後に、六角形の花粉光環という言葉を使っているが、 明るい部分の個数は内側から外側にむかって 6, 8, 10 と増える。 六角形より外側は暗くて観測が難しいいため、 一番内側の六角形で多重多角形構造を代表している。

解析プログラム(ソースコード)#

# -*- encoding: utf-8 -*-

method = "Fresnel Kirchhoff"
import numpy as np
import matplotlib.pyplot as plt
DEG = np.pi/180

qz = 1 # [m] 開口面から像面までの距離 >> (π/λ)*R^2 = 1e-3[m]
q_ang = 7*DEG # [rad] 像面の範囲に対応する角度(片側)

lam = 720e-9 # [m] 波長 (紫:400, 青:490, 緑:550, 赤:720 nm)

W_MARGIN = 1.4 # 計算範囲 (半径相対, Fresnel Kirchhoff の場合、開口部と同程度
N_SEP = 100 # O(N^4) なので 100 くらいが限界

def norm(x):
    a = np.abs(x)
    return a / a.max()
    
def mapxy(f, x, y):
    xx, yy = np.meshgrid(x, y)
    zz = np.vectorize(f)(xx, yy)
    return xx, yy, zz
    
def _f(x1, y1, x2, y2):
    def _line(x): # (x1, y1), (x2, y2) を通る直線を表す関数
        return y1 + (x - x1)*(y2 - y1)/(x2 - x1)
    return _line
    
def gen_shape_cedar(pr, k_cut, h_papilla, w_papilla):
    """
    Parameters
    ----------
    pr, float : [m] 開口(花粉)半径 (スギ 15e-6)
    w_papilla, float : [-]  パピラ幅(三角の底辺, 半径相対) typ. 0.2～0.4
    h_papilla, float : [-]  パピラ高さ(半径相対) typ. 0.4
    k_cut, float : [-] 円を切り取る高さ(半径相対) (1で切り取りなし, 0 で半円)
    
    Returns
    -------
    shape_cedar, callable : 開口部の形状を表す関数
    
    """
    def shape_cedar(px, py):
        w2 = pr*w_papilla # [m] パピラの幅（三角の底辺)
        y1 = pr*k_cut # [m] 円を切り取る高さ(球部の上端)
        y2 = _f(0, pr*(k_cut + h_papilla), -w2/2, y1)(px) # パピラの左斜線
        y3 = _f(0, pr*(k_cut + h_papilla),  w2/2, y1)(px) # パピラの右斜線
        area_list = [
            (py < y1) and (px**2 + py**2 < pr**2), # 欠けた円
            (py >=y1) and (py < y2) and (py < y3), # papilla
        ]
        return any(area_list)
    return shape_cedar
    
def gen_fkd_pixel(pxx, pyy, puu):
    def fkd_pixel(qx, qy):
        k = 2*np.pi/lam # 波数
        r = np.sqrt((qx - pxx)**2 + (qy - pyy)**2 + qz**2)
        return qz/(1.0j*lam)*(puu*np.exp(1.0j*k*r)/r**2).sum()
    return fkd_pixel
    
def extent(xx, yy):
    # imshow の origin = 'lower' と対
    return np.array([xx[0][0], xx[0][-1], yy[0][0], yy[-1][0]])
    
def my_imshow(ax, zz, ext, ttl, xl, yl, **kw):
    ax.imshow(zz, extent=ext, origin='lower', **kw)
    ax.set_title(ttl)
    ax.set_xlabel(xl)
    ax.set_ylabel(yl)
    
def marker(ax, y_fact):
    """
        y_fact = 1.1 # やや縦長 (K さん 2022年4月5日, 熟した花粉?)
                 1.2 # はっきり縦長 (A さん 2022年3月15日, 未熟な花粉?)
    """
    rnc = [ # 半径, 角数, 記号
          (2.3, 6, 'A'),
          (3.8, 8, 'B'),
          (5.0, 10, 'C')
    ]
    
    
    for (r, n, c) in rnc:
        for j in range(n):
            th = -(360*j/n - 90)*DEG
            x = r*np.cos(th)
            y = r*np.sin(th)*y_fact
            ax.text(x, y, '%s%d' % (c, j + 1), color='k', ha='center', va='center')

def sim(pr, k_cut, h_papilla, w_papilla, flg_marker, y_fact):
    """
    pr, k_cut, h_papilla, w_papilla : gen_shape_cedar() 参照。
    """
    
    # 開口面
    shape_cedar = gen_shape_cedar(pr, k_cut, h_papilla, w_papilla)
    p_ = np.linspace(-pr*W_MARGIN, pr*W_MARGIN, N_SEP)
    pxx, pyy, puu = mapxy(shape_cedar, p_, p_)
    
    # 像面
    fkd_pixel = gen_fkd_pixel(pxx, pyy, puu)
    q_ = np.linspace(-q_ang*qz, q_ang*qz, N_SEP)
    qxx, qyy, quu = mapxy(fkd_pixel, q_, q_)
    
    # 作図
    fig, axes = plt.subplots(1, 2, figsize=(9, 5))
    ax = axes[0]
    sp = 'pr:%.1fμm   k_cut:%.1f   h:%.1f   w:%.1f' % (pr/1e-6, k_cut, h_papilla, w_papilla)
    my_imshow(ax, puu, extent(pxx, pyy)/1e-6,
                'Shape of Pollen \n %s' % (sp), 'Xp [μm]', 'Yp [μm]', cmap='gray')
    
    ax = axes[1]
    my_imshow(ax, np.log10(norm(np.abs(quu)**2)), extent(qxx, qyy)/qz/DEG, 
                'Pollen Corona ($|U|^2$)\n $\lambda$=%.0fnm' % (lam/1e-9), 'Xq [deg]', 'Yq [deg]', cmap='jet', vmin=-3.5, vmax=-1)
    
    if flg_marker:
        marker(ax, y_fact)
    
    fig.text(0.95, 0.95, 'model=%s (N:%d M:%.1f)' % (method, N_SEP, W_MARGIN), fontsize=14, ha='right')
    
sim(pr=15e-6, k_cut=0.6, h_papilla=0.4, w_papilla=0.4, flg_marker=True, y_fact=1.2) # 未熟
sim(pr=15e-6, k_cut=0.8, h_papilla=0.4, w_papilla=0.4, flg_marker=True, y_fact=1.1) # 熟した
sim(pr=15e-6, k_cut=0.6, h_papilla=0.0, w_papilla=0.4, flg_marker=False, y_fact=0.0) # パピラなし (w_papilla > 0)

plt.show()
